# 캡스톤디자인 Team 2 — AI 기반 운동 루틴 관리 앱

> AI 챗봇이 사용자의 신체 정보와 목표에 맞춰 맞춤형 운동 루틴을 추천하고 관리해주는 모바일 앱

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 회원가입 / 로그인 | 기본 정보 + 신체 정보 기반 가입, 카카오 로그인 지원 |
| 메인 화면 | 내 루틴 목록 + 대시보드 요약 |
| AI 챗봇 | 운동 루틴 추천 및 질의응답 |
| 루틴 생성 | AI 추천 루틴 선택 및 저장 |
| 루틴 상세 | 일차별 운동 구성 확인 |
| 아카이브 | 저장된 루틴 목록 관리 |
| 운동 기록 | 완료한 운동 기록 |
| 달력 | 운동 일정 캘린더 뷰 |
| 알림 | 운동 알림 설정 |
| 마이페이지 | 내 정보 및 설정 |

---

## 화면 구성 (Wireframe)

> 피그마 와이어프레임 기반 화면 목록

- **Splash** — 앱 시작 화면
- **로그인** — 아이디/비밀번호 + 카카오 로그인
- **회원가입** — 기본 정보 → 신체 정보 (성별, 나이, 운동경력, 운동 목표, 몸무게)
- **메인** — 내 루틴 카드 + 대시보드 + 챗봇 FAB
- **대시보드** — 운동 통계 요약
- **챗봇** — AI 루틴 추천 채팅
- **루틴 생성** — AI 추천 루틴 목록 + 추천 점수
- **루틴 상세** — 1일차 / 2일차 / 3일차 상세 운동 구성
- **아카이브** — 저장된 루틴 목록
- **운동 기록** — 날짜별 운동 완료 기록
- **달력** — 월간 운동 캘린더
- **알림** — 알림 목록 및 설정
- **마이페이지** — 프로필 및 설정
