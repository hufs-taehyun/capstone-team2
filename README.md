# 캡스톤디자인 Team 2 — SciFit Sync

> 논문 기반 AI가 사용자의 헬스장 기구·신체 정보·운동 목표에 맞춰 최적의 운동 루틴을 추천하고 관리해주는 모바일 앱

---

## 주요 기능

| 기능 | 설명 |
|------|------|
| 회원가입 / 로그인 | 기본 정보 + 신체 정보 기반 가입, 카카오 로그인 지원 |
| 헬스장 설정 | GPS 기반 헬스장 검색 및 등록 |
| 기구 설정 | 헬스장 보유 기구 선택 (브랜드별 필터) |
| 1RM 설정 | 벤치프레스 / 스쿼트 / 데드리프트 / 오버헤드프레스 최대 중량 입력 |
| 홈 | 내 루틴 카드 + 대시보드 (볼륨 / 세션 / 스트릭 통계) |
| AI 루틴 생성 | 목표·부위·세션시간·부상 부위 선택 → 논문 데이터 기반 AI 루틴 생성 |
| 루틴 상세 | 운동 목록 확인 (종목·브랜드·근육 활성화·논문 근거) |
| 운동 기록 | 세트 / 중량(kg) / 횟수 / RPE 입력 + 휴식 타이머 |
| 챗봇 | AI 운동 질의응답 |
| 분석 | 운동 볼륨·데이터 분석 |
| 기록 | 날짜별 운동 완료 기록 |
| 아카이브 | 저장된 루틴 목록 관리 |
| 알림 | 운동 알림 목록 |
| 마이페이지 | 프로필 및 설정 |

## 화면 구성 (Wireframe)

> 피그마 와이어프레임 기반 화면 목록 — [Figma에서 보기](https://www.figma.com/design/lowVkeYctKKWbXx7AYIa6V/%EC%BA%A1%EC%8A%A4%ED%86%A4)

### W-A · 인증 / 온보딩

| ID | 화면 | 설명 |
|----|------|------|
| W-A01 | 스플래시 | 앱 시작 화면 |
| W-A02 | 로그인 | 아이디/비밀번호 + 카카오 로그인 |
| W-A03 | 회원가입 1단계 | 이름 / 아이디 / 비밀번호 / 이메일 / 전화번호 |
| W-A04 | 회원가입 2단계 | 성별 / 나이 / 키 / 몸무게 / 운동경력 / 운동 목표 |
| W-A05 | 비밀번호 재설정 | 이메일로 재설정 링크 발송 |
| W-A06 | 헬스장 설정 | GPS 기반 헬스장 검색 및 선택 |
| W-A07 | 기구 설정 | 헬스장 보유 기구 선택 (브랜드별 필터) |
| W-A08 | 1RM 설정 | 주요 리프트 최대 중량 입력 (kg / lb 전환) |

### W-H · 홈

| ID | 화면 | 설명 |
|----|------|------|
| W-H01 | 홈 메인 | 내 루틴 카드 + 대시보드 (볼륨·세션·스트릭) + AI 인사이트 |
| W-H02 | 루틴 옵션 시트 | 루틴 생성 / 이름 수정 / 삭제 / 취소 |
| — | 알림 | 운동 알림 목록 |

### W-R · 루틴

| ID | 화면 | 설명 |
|----|------|------|
| W-R01 | 루틴 생성 시트 | 홈에서 루틴 생성 진입 |
| W-R02 | 루틴 생성 옵션 | 목표 / 부위 / 세션 시간 / 부상 부위 선택 |
| W-R03 | 루틴 생성 중 | AI 논문 데이터 분석 로딩 화면 |
| W-R04 | 루틴 상세 | 운동 목록 (종목·브랜드·근육 활성화·논문 근거) + 운동 시작 |

### W-W · 운동 기록

| ID | 화면 | 설명 |
|----|------|------|
| W-W01 | 운동 기록 | 세트 / 중량 / 횟수 / RPE 입력 + 동작 가이드 영상 + 휴식 타이머 |

### 하단 내비게이션

`챗봇` · `분석` · `메인` · `기록` · `마이페이지`
